# dotnet-desktop-apps

Portfolio of **.NET desktop applications** — Avalonia UI, business logic, PostgreSQL, role-based access.

Each project lives in its own subfolder.

## Projects

| Folder | Description | Stack |
|--------|-------------|-------|
| [fitness_club](./fitness_club/) | Fitness club management: clients, memberships, group classes, bookings, visits, reports | .NET 8, Avalonia UI 11, EF Core, PostgreSQL |

## How to run any project

1. Open the project folder (see its `README.md`)
2. Set up PostgreSQL and run SQL scripts if needed
3. Configure connection string (env var or local file — never commit secrets)
4. `dotnet run` from the app project folder
